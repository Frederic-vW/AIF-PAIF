Autoinformation (AIF) / Partial Autoinformation (PAIF)Â¶

This notebook contains the code and data examples as used in the article:
F. von Wegner, Partial Autoinformation to Characterize Symbolic Sequences Front Physiol (2018) https://doi.org/10.3389/fphys.2018.01382

Feel free to use, improve, modify and distribute the code as you like. If your modified version still shows resemblance with my original code, please cite the article above.
